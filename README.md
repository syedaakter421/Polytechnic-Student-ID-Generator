# 🇧🇩 Polytechnic Student ID Generator & Verification System

ডিজিটাল পলিটেকনিক স্টুডেন্ট আইডি কার্ড জেনারেটর এবং ভেরিফিকেশন সিস্টেম। এটি বাংলাদেশ কারিগরি শিক্ষা অধিনাধীন যেকোনো পলিটেকনিক ইনস্টিটিউটের ছাত্র-ছাত্রীদের জন্য একটি স্বয়ংক্রিয় ডিজিটাল আইডি কার্ড তৈরি এবং কিউআর কোড (QR Code) ভেরিফিকেশন করার একটি সম্পূর্ণ ফুল-স্ট্যাক ওয়েব অ্যাপ্লিকেশন।

প্লাটফর্মটির অনন্য বৈশিষ্ট্য হলো, এটি ব্যবহার করে শিক্ষার্থীরা নিজেদের নিখুঁত আইডি কার্ড ডাউনলোড করতে পারে, অথরিটি সেই কার্ড ম্যানেজ করতে পারে এবং যেকোনো বহিরাগত কার্ডের সত্যতা নিশ্চিত করতে পারে।

## 🔍 প্রজেক্ট ওভারভিউ (Project Overview)

এই সিস্টেমটি মূলত ৩টি আলাদা অংশকে কেন্দ্র করে তৈরি করা হয়েছে:
1. **পাবলিক পোর্টাল (Public Portal):** যেখান থেকে যেকোনো ব্যক্তি সরাসরি কিউআর কোড স্ক্যান করে অথবা রোল ও রেজিস্ট্রেশন নম্বর ইনপুট দিয়ে যেকোনো শিক্ষার্থীর বর্তমান ও বাস্তব আইডি কার্ড ভেরিফাই করতে পারে।
2. **শিক্ষার্থী প্যানেল (Student Panel):** শিক্ষার্থীরা রেজিস্ট্রেশন করে আইডি কার্ড এক ক্লিকেই PNG এবং PDF হিসেবে মোবাইল অথবা কম্পিউটার থেকে ডাউনলোড করতে পারে।
3. **অ্যাডমিন প্যানেল (Admin Panel):** পলিটেকনিকের অ্যাডমিন কার্ডের সকল ফিল্ড কন্ট্রোল করতে পারে, লোগো, সিগনেচার আপলোড করতে পারে, শিক্ষার্থীর স্ট্যাটাস Approve/Reject করতে পারে এবং আইডি কার্ডগুলো একসাথে ZIP ফাইল আকারে এক্সপোর্ট করতে পারে।

## 🚀 প্রধান বৈশিষ্ট্যসমূহ (Core Features)

### ১. স্টুডেন্ট রেজিস্ট্রেশন ও স্মার্ট ভ্যালিডেশন (Sign Up & Validation)
রজিস্ট্রেশন সময় যেন কোনো ভুয়া ডেটা ইনপুট না হতে পারে সেজন্য সর্বোচ্চ স্তরের ক্লায়েন্ট ও সার্ভার সাইড ভ্যালিডেশন যুক্ত করা হয়েছে:
* **মোবাইল নম্বর ভ্যালিডেশন:** মোবাইল নম্বর ও অভিভাবকের মোবাইল নম্বর অবশ্যই `০১` (01) দিয়ে শুরু হতে হবে এবং ইংরেজি সংখ্যায় ঠিক **১১ ডিজিট** হতে হবে। অন্যথা হলে `The phone number is invalid` এরর প্রদর্শন করবে।
* **রোল নম্বর ভ্যালিডেশন:** রোল নম্বরটি অবশ্যই সঠিক ইংরেজি সংখ্যায় **৬ ডিজিটের** হতে হবে।
* **রেজিস্ট্রেশন নম্বর ভ্যালিডেশন:** রেজিস্ট্রেশন নম্বরটি অবশ্যই সঠিক ইংরেজি সংখ্যায় **১০ ডিজিটের** হতে হবে।
* **সিলেক্টর সিস্টেম:** "Shift" নির্বাচনের জন্য স্ট্যান্ডার্ড ড্রপডাউন (১ম/২য় শিফট) এবং রক্তের গ্রুপ নির্ভুল ইনপুট নেয়ার জন্য ডেডিকেটেড **Blood Group Selector**।

### ২. শক্তিশালী আইডি কার্ড জেনারেটর (Digital ID Card Engine)
* **মোবাইল অপ্টিমাইজড ডাউনলোড (CORS Safe):** আইডি কার্ড জেনারেট করার জন্য কোনো এক্সটার্নাল ইমেজের সোর্স লিংক ব্যবহার করা হয়নি। ডাটাবেজে ইমেজগুলো Base64 ফর্ম্যাটে সুরক্ষিত থাকে, যার ফলে কোনো রকমের Cross-Origin (CORS) ইস্যু বা ব্ল্যাঙ্ক ইমেজ ডাউনলোড সমস্যা হয় না।
* **মোবাইল রেন্ডারিং বাগ ফিক্স:** বিশেষ অফস্ক্রিন পজিশনিং ও মোবাইল ব্রাউজারের ক্যানভাস মেমোরি লিমিট (Canvas Memory Limit) ফিক্স করতে ডাইনামিক Pixel Ratio এবং রেন্ডারিং ডিলে (Delay) যুক্ত করা হয়েছে।
* **হাই-কোয়ালিটি এক্সপোর্ট:** শিক্ষার্থী তার পছন্দের আইডি কার্ড অত্যন্ত ঝকঝকে PNG এবং উচ্চ-মানের PDF হিসেবে ডাউনলোড করতে পারে।

### ৩. পাবলিক ভেরিফিকেশন সিস্টেম (Verification Portal)
* আইডি কার্ডের ব্যাক পার্টে (Back Side) একটি ইউনিক **QR Code** স্বয়ংক্রিয়ভাবে জেনারেট হয়।
* এই কিউআর কোড স্ক্যান করলেই ভেরিফিকেশন পোর্টালের লাইভ ডেটা চলে আসে, যা জাল আইডি কার্ড তৈরি সম্পূর্ণ বন্ধ করে দেয়।

### ৪. ডাইনামিক অ্যাডমিন ড্যাশবোর্ড (Super Admin Dashboard)
* **রিয়েল-টাইম স্ট্যাটিস্টিকস:** সর্বমোট শিক্ষার্থী, পেন্ডিং রেজিস্টার্ড, ভেরিফাইড এবং রিজেক্টেড স্টুডেন্টের লাইভ গ্রাফিক্যাল বাটন।
* **অ্যাডভান্সড ফিল্টারিং:** রোল, টেকনোলজি, সেমিস্টার, শিফট বা নামের ওপর ভিত্তি করে সেকেন্ডের মধ্যে শিক্ষার্থী খোঁজার সিস্টেম।
* **বাল্ক অ্যাকশন (Bulk Actions):** একাধিক শিক্ষার্থীকে একসাথে ওয়ান-ক্লিকে Approve বা Reject করার সুবিধা।
* **বাল্ক জিপ ডাউনলোড (Bulk Registration Export):** নির্বাচিত সকল শিক্ষার্থীর আইডি কার্ড একসাথে একক ক্লিকেই ১টি **ZIP ফাইলে** হাই-কোয়ালিটি জেনারেট করে ডাউনলোড করার অ্যাডভান্সড ফিচার।
* **ইনস্টিটিউট কনফিগারেশন:** ইনস্টিটিউটের লোগো, অ্যাডমিনের ডিজিটাল সিগনেচার, এবং প্রোফাইল ইনফরমেশন যেকোনো সময় অ্যাপ থেকেই ডাইনামিকালি অ্যাডমিন পরিবর্তন করতে পারে।

## 🛠️ ব্যবহৃত প্রযুক্তিসমূহ (Technologies Used)

সফটওয়্যারটির নির্ভরযোগ্যতা ও পারফরম্যান্স নিশ্চিত করার জন্য আধুনিক ও শক্তিশালী টেক-স্ট্যাক ব্যবহার করা হয়েছে:

* **ফ্রন্টএন্ড (Frontend):**
  * **React 18 & Vite:** আল্ট্রা-ফাস্ট লোডিং স্পিড এবং মডার্ন সিঙ্গেল পেজ অ্যাপ্লিকেশন (SPA) বানানোর জন্য।
  * **TypeScript (TS):** টাইপ-সেফ কোড ডেভলপমেন্টের মাধ্যমে সকল ডেটা টাইপ হ্যান্ডেল করার জন্য।
  * **Tailwind CSS:** সুন্দর, রেসপন্সিভ এবং বাংলাদেশ সরকারের লোহিত-সবুজ আইডেন্টিটি প্যালেটে (Government theme) ওয়েবসাইট সাজানোর জন্য।
  * **Framer Motion (`motion/react`):** দৃষ্টিনন্দন ড্যাশবোর্ড ট্রানজিশন এবং সাইনআপ ফর্ম অ্যানিমেশনের জন্য।
  * **html-to-image:** DOM এলিমেন্টগুলোকে নির্ভুল ক্যানভাসে কনভার্ট করে ইমেজ জেনারেটের জন্য।
  * **jspdf:** সরাসরি ক্লায়েন্ট সাইড থেকে ডাইনামিক স্টুডেন্ট আইডি PDF স্ক্রিপ্টিং করার জন্য।
  * **lucide-react:** চমৎকার ও আকর্ষণীয় আধুনিক ভেক্টর আইকন লাইব্রেরি।

* **ব্যাকএন্ড (Backend):**
  * **Node.js & Express (TypeScript-driven):** দ্রুত ও স্কেলেবল এপিআই (API) সার্ভারের কাজ সম্পন্ন করতে।
  * **Multer:** ইমেজ প্রসেস ও Base64 কনভার্ট করে দ্রুত ডাটাবেজে স্টোর করার জন্য।
  * **jsonwebtoken (JWT):** সুরক্ষিত সেশন এবং অথেনটিকেশনের জন্য।

* **ডাটাবেজ (Database):**
  * **Supabase / PostgreSQL:** হাই-স্পীড রিলেশনাল ডাটাবেজ সিস্টেম।

## 🛡️ নিরাপত্তা বৈশিষ্ট্যসমূহ (Security Controls)

লাইভ প্রোডাকশন সার্ভারে ডিপ্লয়মেন্টের পর ওয়েবসাইটকে যেকোনো ক্ষতিকর আক্রমণ থেকে বাঁচতে শক্তিশালী নিরাপত্তা বলয় সংযুক্ত করা হয়েছে:

1. **এক্সপ্রেস রেট লিমিটিং (Express Rate Limiter):**
   * সার্ভারে brute-force বা DDoS আক্রমণ ঠেকাতে `express-rate-limit` নিযুক্ত করা হয়েছে।
   * প্রতি ১৫ মিনিটে কোনো নির্দিষ্ট আইপি থেকে ২০০ বারের বেশি রিকোয়েস্ট পাঠানো ব্লক হয়ে যাবে। এর ফলে সার্ভার স্প্যামিং থেকে সম্পূর্ণ মুক্ত থাকে।
2. **নিরাপদ ডাটা প্রবাহ ও কনসোল হাইডিং:**
   * ফ্রন্টএন্ডে শিক্ষার্থীদের ব্যক্তিগত তথ্যের নিরাপত্তা রক্ষার্থে কোনো সিক্রেট কী (API Keys) বা সেন্সিটিভ ডেটা লিক হতে পারে এমন কোনো `console.log` যুক্ত রাখা হয়নি।
   * এর ফলে লাইভ হোস্টিংয়ের পর হ্যাকাররা বা সাধারণ ব্যবহারকারীরা ইন্সপেক্ট এলিমেন্টের কনসোল ট্যাব থেকে কোনো ডেডিকেটেড কুয়েরি বা ডেটা ব্রাউজ করতে পারবে না।
3. **JWT ভিত্তিক সেশন সুরক্ষিতকরণ:**
   * প্রতিটি স্টুডেন্ট এবং অ্যাডমিন রাউটকে সুরক্ষিত করতে JWT মডিউল ব্যবহার করা হয়েছে যা অননুমোদিত অ্যাক্সেস সম্পূর্ণ প্রতিরোধ করে।
4. **CORS সিকিউরিটি:**
   * সকল মিডিয়া ও ইমেজকে সরাসরি Base64 স্টোরেজ ফরম্যাটে রূপান্তর করে রাখা হয়। তাই ক্লায়েন্ট থেকে ডাউনলোড করার সময় কোনো থার্ড-পার্টি প্লাগইন বা ওরিজিনাল সার্ভার প্রক্সির সাহায্য ছাড়াই দ্রুত কাজ করে।

## 📐 ইউজার ইন্টারফেস ও আর্কিটেকচার (UI & Architecture)

প্রকল্পটিতে বাংলাদেশ টেকনিক্যাল এডুকেশনের থিম মাথায় রেখে ডিপ মেটালিক স্লেট গ্রে, রয়্যাল গোল্ডেন গোল্ড এবং গর্ভমেন্ট গ্রিন-রেড বর্ডার কম্বিনেশন ব্যবহার করা হয়েছে। 

* **Desktop-First Precision & Mobile Adaptability:** কম্পিউটারে বড় পর্দায় যেভাবে কার্ডটি নির্ভুল সাইজে দেখা যায়, মোবাইলেও তার ভিউ ঠিক রাখার জন্য বিশেষ CSS রুলস মেনে কোড স্ট্রাকচার সাজানো হয়েছে।
* **No Unnecessary Metadata UI:** অ্যাপ্লিকেশনটিতে কোনো রোবটিক সিস্টেম পিন বা আইফ্রেম ট্র্যাকিং ট্যাগ ব্যবহার করা হয়নি; সম্পূর্ণ লেআউটটি অত্যন্ত মানবিক ও ইউজার ফ্রেন্ডলি।

## 💻 ইনস্টলেশন ও রান করার নিয়ম (Installation Guide)

আপনার লোকাল সিস্টেমে প্রজেক্টটি চালু করতে নিচের ধাপগুলো অনুসরণ করুন:

১. প্রজেক্ট ফাইলটি ক্লোন করুন অথবা ডাউনলোড জিপ টি এক্সট্র্যাক্ট করুন।
২. আপনার টার্মিনালে প্রবেশ করে ডিপেন্ডেন্সি ইনস্টল করার জন্য নিচের কমান্ডটি চালান:
```bash
npm install
```
৩. ডেভেলপমেন্ট মোডে রান করার জন্য লিখুন:
```bash
npm run dev
```
৪. প্রোডাকশন বিল্ড তৈরি করার জন্য নিচের কমান্ডটি ব্যবহার করুন:
```bash
npm run build
```
৫. সফলভাবে বিল্ড হওয়ার পর প্রোডাকশন রান করুন:
```bash
npm run start
```

---

*তৈরি করেছেন আপনার টিম। এই বিশ্বস্ত ও প্রফেশনাল আইডি জেনারেটরে কারিগরি শিক্ষার্থীদের প্রযুক্তি দক্ষতা এবং সচ্ছতা আরো বৃদ্ধি পাবে।*
